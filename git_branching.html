• What git command creates a branch? git checkout -b
  • What is the difference between a fast-forward and recursive merge? FF merges take place if no commits have occurred on original branch while new branch is being worked on
  • What git command changes to another branch? git checkout
  • What git command deletes a branch? git branch -d [NAME-OF-BRANCH]
